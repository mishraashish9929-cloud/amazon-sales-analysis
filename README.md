# Amazon E-commerce Sales Analysis (SQL & Excel)

## Overview
This project analyzes 128k+ Amazon sales transactions to identify revenue trends, geographical performance, and top-selling products.

## Tools Used
- **SQL:** Data cleaning, aggregating revenue, and regional analysis.
- **Python/Pandas:** Data preprocessing and automation.
- **Excel/Matplotlib:** Dashboard creation and visualizations.

## Key Insights
- **Total Revenue:** ₹ 78.6 Million.
- **Top State:** Maharashtra.
- **Bestseller:** Style JNE3797.
- **Peak Period:** April 2022.

## Files
- `Amazon Sale Report.csv`: Original Dataset
- `queries.sql`: SQL scripts for analysis
- `dashboard_charts.png`: Visual representations
